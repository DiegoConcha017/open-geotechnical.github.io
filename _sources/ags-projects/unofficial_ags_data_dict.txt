Unofficial AGS Data Dict
===============================

- The AGS data dictionary is available in JSON format
- This is unofficial and not guaranteed to be up to date
- Visit upstream http://www.agsdataformat.com for correct data

Download
------------

Point your browser at 

- https://open-geotechnical.github.io/data/ags4.min.json

The file contains the following keys:

- **groups** - the ags groups and their headings
- **abbrs** - the abbreviations
- **units** - units and descriptions
- **data_types** - the data types
- **examples** - Example ags files
