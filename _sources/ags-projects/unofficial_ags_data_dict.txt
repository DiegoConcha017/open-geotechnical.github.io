Unofficial AGS4 Data Dict
===============================

The Unofficial AGS data dictionary is the complete spec in a single minified json file.

.. warning::

    - This is unofficial and not guaranteed to be up to date.
    - Visit upstream website at `AgsDataFormat.com/datatransferv4 <http://www.agsdataformat.com/datatransferv4/intro.php>`_ for latest correct data.

Download
------------

- https://open-geotechnical.github.io/data/ags4.min.json

Structure
-------------------

The json file contains the following keys:

- **groups** - the ags groups and their headings
- **abbrs** - the abbreviations
- **units** - units list
- **data_types** - the data types
