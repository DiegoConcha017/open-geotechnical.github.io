Notes - 3
-------------------------
- **PIPE** is to be used to transfer pipe work lengths installed in an exploratory hole to provide instrumentation access.
- A data row is included for each pipe length/type/diameter combination in a hole.
- **PIPE_TYPE** would typically be an abbreviation and define the graphics used to represent the installation pipe work on reports or diagrams.
